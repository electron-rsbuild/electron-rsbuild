# create-electron-rsbuild TODO

Create a new Rsbuild project.

Using `npm create`:

```bash
npm create electron-rsbuild@latest
```

Using CLI flags:

```bash
npx create-electron-rsbuild --dir my-project --template react

# Using abbreviations
npx create-electron-rsbuild -d my-project -t react
```
