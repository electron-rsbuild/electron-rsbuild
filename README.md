# electron-rsbuild

Next generation Electron build tooling based on Rsbuild.

## cmd-lines

```js
{
  "scripts": {
    "start": "electron-rsbuild preview", //预览生产构建
    "dev": "electron-rsbuild dev",// 开发服务和electron 程序
    "build": "electron-rsbuild build",// build 构建
  }
}

```
